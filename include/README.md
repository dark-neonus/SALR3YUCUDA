# include/

Public C header files defining the interfaces shared across all build targets (CPU and future CUDA).

| Header | Purpose |
|--------|---------|
| `config.h` | Configuration parser interface and `SimConfig` structure |
| `grid.h` | Computational grid definition |
| `potential.h` | SALR 3-Yukawa pair potential interface |
| `solver.h` | Iterative DFT solver interface |
| `math_utils.h` | Vector and matrix arithmetic operations |
| `io.h` | File I/O helper functions |
